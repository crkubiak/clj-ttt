> TTT written in the Clojure language.  Uses speclj testing and cloverage.

## Install
Clone or fork the repo.

## Usage

In /src/clj_ttt directory: 
```sh
clj core.clj 
```

## Run tests
In root directory
```sh
lein spec or
lein spec -a for continuous testing
lein cloverage for coverage statistics
```
